## NordVPN
```
I had the same issue, so came searching. I haven't tried the wireguard solution suggested by another commenter, but I found a solution for the openvpn version...
It seems that NordVPN recently changed their Auth process.
To fix it follow these steps:
 - Log in to Nord's website with your normal creds
 - Under accounts, services, go to NordVPN
 - At the bottom of the page there is a button "Set up NordVPN Manually"
 - You get a verification email, enter the code
 - Copy the new credentials into gluten and it works.
```
